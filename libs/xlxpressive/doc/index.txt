mpllibs::xlxpressive
==================
István Siroki

Xlxpresive is a working implementation of a compile-time string based
interface for static Xpressive. This solution gives a macro called
+MPLLIBS_REGEX+ which can be used like a function which has a string
argument; the regular expression. The static Xpressive object which corresponds to the specified string will be generated in compile-time.
