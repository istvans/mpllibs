mconcat
=======
Abel Sinkovics

Synopsis
--------

--------------------------------------
template <class MonadTag, class L>
struct mconcat
{
  // unspecified
};
--------------------------------------


Description
-----------

+mconcat+ is a metafunction for the +mconcat+ operation of the
link:monoid.html[monoid] typeclass.

Header
------

--------------------------------------
#include <mpllibs/metamonad/mconcat.hpp>
--------------------------------------

Expression semantics
--------------------

For any +monad_tag+ tag and +l+ list:

--------------------------------------
mconcat<monad_tag, l>
--------------------------------------

is equivalent to

--------------------------------------
boost::mpl::apply_wrap2<monoid<monad_tag>::mconcat, l>
--------------------------------------

Example
-------

--------------------------------------
using boost::mpl::list_c;
using boost::mpl::list;

typedef
  mconcat<
    list_tag,
    list<
      list_c<int, 0, 1, 2, 3, 4>,
      list_c<int, 5, 6>,
      list_c<int, 7, 8, 9>
    >
  >::type
  list_0_to_10;
--------------------------------------
